# 2026-05-10 Changelog

## 2026-05-10 - Clipboard Glass and Pin Behavior

This pass continued the native macOS polish around clipboard history and the translation island.

### Implemented

- Kept the right side of the closed island wired to open clipboard history, while the left side remains the hover/click translation entrance.
- Updated the translation island pin behavior so automatic pointer-leave and outside-click collapse respect the pinned state.
- Preserved Escape as an explicit way to dismiss the island even when pinned.
- Added local JSON persistence for clipboard history under Application Support so history survives app restarts.
- Simplified clipboard history into a borderless floating panel, removed list shortcut and pin controls, and made Escape close the panel reliably through a local key monitor.
- Tuned clipboard typography lighter and aligned Settings dark mode with the blue-gray glass palette used by the island and clipboard surfaces.
- Added a configurable global hotkey for opening clipboard history, defaulting to `Option-C`.

### Documentation

- Updated README capability notes for the expanded clipboard history surface.
- Replaced stale clipboard-history follow-up language with the current remaining limitation: rich text is not modeled separately yet.
